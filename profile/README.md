# turnie

 ### すれちがい通信デバイス「turnie」

 ```デモプログラムなのでturnie同士をかなり近づけないとデータ交換がされないようになっています。是非デバイス同士を近づけてみて下さい。```

- turnie-app
  - iOSアプリケーション
  - テキストやピクセルアートをデバイスに送信
- turnie-device
  - turnie本体のファームウェア
  - iOSアプリからの書き込みを受信・turnie同士での通信
- turnie-garber
  - turnie本体の基板データ
  - 必要部品および回路図
