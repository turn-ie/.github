# turnie

### すれちがい通信デバイス「turnie」

- turnie-app
  - iOSアプリケーション
  - テキストやピクセルアートをデバイスに送信
- turnie-device
  - turnie本体のファームウェア
  - iOSアプリからの書き込みを受信・turnie同士での通信
- turnie-garber
  - turnie本体の基板データ
  - 必要部品および回路図
